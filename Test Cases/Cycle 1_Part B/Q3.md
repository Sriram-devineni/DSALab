### Test Case 1

Input

```
5(3(1)(2))(6)
p 2 5
a 6
```

Output

```
2 3 5
6 5
```

Explanation for output

```
p:followed by two distinct integer values is to print the key values of the nodes that are in the given range 2 3 5.
a:prints the ancestors 6 5.

```
